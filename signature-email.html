Comment installer cette signature dans Gmail

  1. Ouvre Gmail → Paramètres (roue crantée) → Voir tous les paramètres
  2. Onglet Général, descends jusqu’à Signature
  3. Clique sur Créer, nomme-la « balise-ia »
  4. Clique dans le bloc navy ci-dessous, fais Ctrl+A pour tout sélectionner dans le bloc, puis Ctrl+C
  5. Colle avec Ctrl+V dans l'éditeur de signature Gmail
  6. Enregistre les modifications en bas de page
